# infraNode

Repositorio de infraestructura Node.js para proyectos DevOps/MLOps.

## Descripción

Proyecto de microservicios con Node.js, Docker y configuración de infraestructura automatizada.

## Tecnologías

- Node.js
- Docker
- Express
- PostgreSQL/MongoDB

## Instalación

```bash
npm install
```

## Uso

```bash
npm start
```
